# summary-maker

### Delivers a summary of a specific topic on google docs

![](/images/usage.gif)
![](/images/summary.png)

## How to Use

Clone the repository

```bash
git clone https://github.com/rafaelaraujo13/summary-maker.git

cd summary-maker
```

Install the dependencies

```bash
pip install -r requirements.txt
```

Run the app

```bash
python main.py
```
